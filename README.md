# Number row characters
- Shift + 1 = ! (Exclamation Mark)
- Shift + 2 = @ (At Symbol)
- Shift + 3 = \# (Hash or Pound Symbol)
- Shift + 4 = $ (Dollar Sign)
- Shift + 5 = % (Percent Sign)
- Shift + 6 = ^ (Caret)
- Shift + 7 = & (Ampersand)
- Shift + 8 = * (Asterisk)
- Shift + 9 = ( (Left Parenthesis)
- Shift + 0 = ) (Right Parenthesis)
